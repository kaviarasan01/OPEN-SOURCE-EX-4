# OPEN-SOURCE-EX-4

## Create a collaborative directory /common/admin with the following characteristics:

a) Group ownership of /common/admin is admin.

b) The directory should be readable,writable and accessible to members of admin, but not to any other user. (It is understood that root has access to all files and directories on the system.)

c) Files created in /common/admin automatically have group ownership set to the admin group.
